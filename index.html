Purpose

Scikit-learn Central is a community resource dedicated to serving the broader scientific Python and machine learning ecosystem. It’s curated, hosted, and governed by Probabl, with contributions from the community at large.

The catalog is open to any package that belongs to the scikit-learn open-source ecosystem and carries a true open-source license. Default rankings reflect usage signals and ecosystem fit; Probabl-maintained packages may appear prominently as Probabl is the steward of this resource.

The use cases are curated and forward-looking, highlighting how data scientists are using libraries in the scikit-learn ecosystem to tackle real-world machine learning problems.

Ranking Methodology

Each ecosystem package is assigned a Fit Score (0–100) displayed as a chip in the top-right corner of its card. Hovering over the chip reveals the three underlying sub-scores.

Stars ⅓
GitHub star count on a logarithmic scale. The package with the most stars in the catalog scores 100%; others are normalised relative to that maximum. The log scale dampens the outsized effect of very popular packages.
Monthly Downloads ⅓
PyPI monthly download count, also on a logarithmic scale. Reflects real-world adoption and active usage in production codebases, normalised to the catalog maximum.
Use Cases ⅓
Number of curated use cases in which the package appears, on a linear scale. A higher count signals broader applicability across real-world ML problems.

Fit Score = (Starslog + Downloadslog + UseCaseslinear) ÷ 3

Scores are computed dynamically from live catalog data and renormalised whenever the catalog is updated. The catalog is deliberately neutral — scikit-learn core is excluded from the normalisation pool since it is displayed separately as the ecosystem foundation.
